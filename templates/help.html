Descarga o envía por email

La herramienta acepta los siguientes formatos:

  • CSV (.csv): Archivos de valores separados por comas
  • Excel (.xlsx): Archivos de Excel modernos
  • Excel (.xls): Archivos de Excel antiguos
Importante: El tamaño máximo del archivo es 16MB

La herramienta realiza las siguientes operaciones de limpieza automáticamente:

  • Eliminación de duplicados: Remueve registros idénticos
  • Limpieza de filas vacías: Elimina filas completamente vacías
  • Normalización de fechas: Convierte fechas a formato estándar
  • Conversión de tipos: Identifica y convierte columnas numéricas
  • Manejo de valores faltantes: Rellena valores vacíos apropiadamente

La herramienta genera automáticamente los siguientes tipos de gráficos:

  • Gráficos de barras: Para distribuciones de valores numéricos
  • Gráficos de pastel: Para mostrar proporciones por categorías
  • Gráficos de línea: Para series temporales
  • Histogramas: Para distribuciones de frecuencias
  • Matrices de correlación: Para relaciones entre variables
Tip: Los gráficos se seleccionan automáticamente según el tipo de datos detectado

Para configurar el envío por email, sigue estos pasos:

1. Crear archivo .env
EMAIL_USER=tu_correo@gmail.com
EMAIL_PASSWORD=tu_contraseña_de_aplicacion
SMTP_SERVER=smtp.gmail.com
SMTP_PORT=587
2. Configurar Gmail (si usas Gmail)
  • Ve a Configuración de tu cuenta Google
  • Seguridad → Contraseñas de aplicación
  • Crea una nueva contraseña de aplicación
  • Usa esa contraseña en el archivo .env
Seguridad: Nunca uses tu contraseña normal de Gmail. Siempre usa contraseñas de aplicación.

Si el email no llega, verifica lo siguiente:

  • Revisa la carpeta de spam: Los emails pueden ir a spam
  • Verifica las credenciales: Asegúrate de que EMAIL_USER y EMAIL_PASSWORD estén correctos
  • Contraseña de aplicación: Para Gmail, usa una contraseña de aplicación, no tu contraseña normal
  • Configuración SMTP: Verifica que SMTP_SERVER y SMTP_PORT estén correctos
  • Firewall/Antivirus: Algunos programas pueden bloquear el envío
Tip: Puedes probar la conexión de email desde la página principal

El reporte Excel incluye las siguientes hojas:

  • Resumen Ejecutivo: Estadísticas principales y hallazgos clave
  • Datos: Dataset completo con limpieza aplicada
  • Gráficos: Visualizaciones automáticas con datos
  • Análisis: Estadísticas descriptivas detalladas
Característica: Los gráficos son interactivos y se actualizan automáticamente con los datos

Solución de Problemas Comunes

Error al subir archivo
  • Verifica que el archivo sea CSV o Excel
  • Asegúrate de que no exceda 16MB
  • Revisa que el archivo no esté corrupto
No se generan gráficos
  • Verifica que haya columnas numéricas
  • Asegúrate de que los datos no estén vacíos
  • Revisa que las columnas tengan nombres claros
Error de email
  • Verifica las credenciales en .env
  • Usa contraseña de aplicación para Gmail
  • Revisa que el puerto 587 esté abierto
Error al descargar
  • Verifica que el reporte se haya generado
  • Revisa los permisos de la carpeta reports
  • Intenta generar el reporte nuevamente
